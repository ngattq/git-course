1. Giải thích các câu lệnh: git commit --amend, git commit --amend
-m”<message>”, git restore --staged <file>, git reset HEAD~1.
 - git commit --amend : Dùng để sửa đổi commit gần nhất.
 - git commit --amend -m”<message>”: Tương tự như git commit --amend, nhưng thay vì mở trình soạn thảo để sửa message, truyền trực tiếp message mới vào qua tùy chọn -m
 - git restore --staged <file>: Đưa từ vùng staging về working
directory
 - git reset HEAD~1.: Đưa từ vùng repository về
working directory (uncommit)
